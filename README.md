# Maksym Varybrus — Landing

Личный лендинг (маркетинг и разбор бизнеса). Тёмная тема, видео в hero, прелоадер, три языка (CS / UA / RU), bounce-бегущая строка.

## Файлы
- `index.html` — сам сайт
- `hero.mp4` — видео для hero (должно лежать рядом с index.html)
- `package.json` — запуск статики на Railway / Node-хостингах
- `.gitignore`

## Локально
Просто открой `index.html` в браузере (держи `hero.mp4` в той же папке).

## Деплой на Railway
1. Залей репозиторий на GitHub.
2. railway.app → New Project → Deploy from GitHub repo → выбери этот репозиторий.
3. Settings → Networking → Generate Domain — получишь публичную ссылку.

Railway сам выполнит `npm install` и `npm start` (раздаёт папку через `serve` на `$PORT`).

## Бесплатная альтернатива — GitHub Pages
Settings → Pages → Deploy from branch → `main` → `/root`. `package.json` для этого не нужен.
